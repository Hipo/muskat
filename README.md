muskat
======
  
AWS resource cleanup tool  
  
Usage:  
    muskat.py show <resource> [<region> --verbose]  
    muskat.py clean <resource> [<region> --verbose]  
    muskat.py (-h | --help)  
  
  
Options:  
    <resource>   all, %s  
    <region>     all, %s  
	-h --help    Show this screen  
    --verbose    Verbose Output
